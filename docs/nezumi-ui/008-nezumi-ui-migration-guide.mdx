# Migration Guide

## Import Migration

Move consumer imports to public leaves:

```tsx
import { Button } from "@packages/ui/components/button"
import { Container } from "@packages/ui/layout"
```

## Token Migration

Replace JavaScript token wrappers with Tailwind utilities or CSS variables:

```tsx
<div className="bg-surface p-16" />
```

```css
.card {
  background: var(--color-surface);
  padding: var(--spacing-16);
}
```

## Package Migration

The UI package owns:

- `src/styles/global.css`
- `src/styles/design-tokens.css`
- `src/styles/tokens/*.css`
- `src/styles/semantic/*.css`
- `src/styles/components/*.css`
- `src/components/*.tsx`
- `src/layout/index.ts`

Do not add a second token source.

## Verification

```bash
pnpm install
pnpm build --filter @packages/ui
pnpm build:packages
```
