# Public API

The public API is the contract between apps and the UI package. It is intentionally granular.

## Correct Imports

```tsx
import { Button } from "@packages/ui/components/button"
import { Box } from "@packages/ui/components/box"
import { Container } from "@packages/ui/components/container"
import { Flex } from "@packages/ui/components/flex"
import { Grid } from "@packages/ui/components/grid"
import { Section } from "@packages/ui/components/section"
import { cn } from "@packages/ui/lib/utils"
```

## Package Exports

```json
{
  "imports": {
    "#components/*": "./src/components/*.tsx",
    "#lib/*": "./src/lib/*.ts"
  },
  "exports": {
    "./globals.css": "./src/styles/global.css",
    "./design-tokens.css": "./src/styles/design-tokens.css",
    "./components/button": "./src/components/button.tsx",
    "./components/box": "./src/components/box.tsx",
    "./components/container": "./src/components/container.tsx",
    "./components/flex": "./src/components/flex.tsx",
    "./components/grid": "./src/components/grid.tsx",
    "./components/section": "./src/components/section.tsx",
    "./layout": "./src/layout/index.ts",
    "./lib/utils": "./src/lib/utils.ts"
  }
}
```

Because `@packages/ui` is internal to this monorepo, package exports point at
source files. Next.js apps keep `transpilePackages: ["@packages/ui"]`, and
TypeScript resolves package imports/exports through `moduleResolution: "Bundler"`.

## Why Granular Exports

- Apps do not depend on atomic folder names.
- Test and build dependency graphs stay small.
- shadcn-generated components can be adapted behind a stable leaf file.
- Internal refactors happen behind `src/components/<name>.tsx` leaf files without consumer churn.

## Adding A Public Component

1. Implement internally where it belongs.
2. Add `src/components/<name>.tsx`.
3. Add the matching source export in `packages/ui/package.json`.
4. Re-export the public component and public types from that leaf.
5. Document the consumer import path.

Example:

```tsx
import { Button } from "@packages/ui/components/button"
```
