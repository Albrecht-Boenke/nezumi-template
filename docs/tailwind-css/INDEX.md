# Tailwind CSS (offline Snapshot)

**Wann diese Sektion lesen?**

- Du brauchst **Utility-Referenz** (Spacing, Farben, Layout, …) oder **Theming / @theme**.
- Du arbeitest an **Dark Mode**, **States** oder **Custom Styles** laut Offizielldoku.
- Du willst **ohne Netz** die gleiche inhaltliche Abdeckung wie auf tailwindcss.com.

**MDX-Hinweis:** Praktisch alle Seiten sind `.mdx`; Code- und Komponentenblöcke stammen aus der Original-Site—normalerweise trotzdem gut lesbar als Text.

Die Seiten entsprechen dem Inhalt unter [tailwindlabs/tailwindcss.com – `src/docs`](https://github.com/tailwindlabs/tailwindcss.com/tree/master/src/docs), hier ohne `src/`-Präfix.

1. **Dateiname ≈ Utility- oder Kapitelslug** auf tailwindcss.com (z. B. `flex`, `theme`, `dark-mode`).
2. IDE-Schnellöffner: `tailwind-css/<begriff>.mdx` — es gibt **keine** vollständige Linkliste in dieser `INDEX.md` (Absicht: Kontext sparen).
3. Nur bei Bedarf **README** oder die Beispielzeilen unten öffnen, dann gezielt die eine Zieldatei.
